# Cluster for Data Engineering

## Overview

This script deploys a cluster in fully distributed mode using Docker as the underlying infrastructure.

## Authors

- Emanoel B. P. Filho
- Raphael R. Silva
- Carlos M. D. Viegas

## How to run it

### Build it

```bash
make
```

### Run it

```bash
docker compose up -d
```
